**An interactive Karate Kido game simulation with automated player movements, pose recognition, and scoring system.**

## ✨ Features

- Real-time Karate Kido gameplay simulation
- Pose detection and movement tracking
- Automated opponent AI
- Scoring and combo system
- Interactive controls
- Visual effects for kicks and punches
- Performance analysis

## 🛠 Technologies Used

- **Engine**: Unity3D
- **Vision**: OpenCV / MediaPipe (for pose estimation)
- **Language**: C#
- **AI**: Rule-based or simple ML for opponent behavior

## 📋 Project Details

- **Project Title**: محاكاة لعبة Karate Kido باستخدام Computer Vision
- **University**: Damascus University - Faculty of Informatics Engineering
- **Course**: Computer Vision
- **Year**: 2024

## 🎯 Main Components

- Character animation and physics
- Pose detection for player input
- Automated opponent movements
- Collision detection and scoring
- Visual feedback system

## 🚀 How to Run

1. Open project in Unity
2. Play the main scene
3. Use keyboard/motion controls for Karate moves
